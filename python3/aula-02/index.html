Variáveis

O que e uma variável?
Uma variável é uma área de memória onde armazenamos um valor qualquer. Esta área é associada a um nome ou identificador.
Tipos de variáveis
str(strings) Uma cadeia de caracteres geralmente utilizada para representar palavras, frases ou textos.
int(inteiros) Números sem virgulas positivos e negativos.
float(flutuantes) Números com decimais(Exemplo: 1.52).
bool(Verdadeiro/Falso) Valore True ou False(1 ou 0).

Codigo Variáveis

# String
nome = "Pedro"
print("Nome: ",nome)
# Inteiros
idade = 17
print("Idade",idade)
# Flutuantes
peso = 55.6
print("Peso",peso)
# booleanos
sexo = False
# Compara
if sexo:
    print(f"Masculino: {sexo}")
else:
    print(f"Feminino: {sexo}")

Codigo Fazer Perfil

# String
nome = str(input("Nome: "))
# Inteiros
idade = int(input("Idade: "))
# Flutuantes
peso = float(input("Peso: : "))
# Str
sexo = str(input("Sexo M para Masculino F para Feminino: "))
print("Nome: ",nome)
print("Idade",idade)
print("Peso",peso)
print("Sexo: ",sexo)
